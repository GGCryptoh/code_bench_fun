Privacy Policy

No Data Collection

Code Bench Fun is a static website with no servers, no accounts, and no data collection. We do not use cookies, analytics, tracking pixels, or advertising. We do not collect, store, or process any information about you or your use of this site.

Your OpenRouter API Key

When you use the One-Shot Bench page, you provide your own OpenRouter API key. This key is stored only in your browser's localStorage and is sent only directly to OpenRouter.ai to run game builds. It never touches our servers (we don't have any). You can clear it anytime by clicking the Clear button on that page or by clearing your browser's site data.

Your Prompts and Builds

Prompts you type on the bench page are sent directly from your browser to OpenRouter and the model providers. These are subject to OpenRouter's and the providers' own privacy policies and terms of service.

Generated Games

Games displayed in the gallery are AI-generated. They run sandboxed in your browser and make no network requests of any kind.

Hosting and Server Logs

This site is hosted on GitHub Pages. GitHub may log standard server access data such as IP addresses and request metadata per their own privacy policy.
